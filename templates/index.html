Welcome to the Flickrflask Example app


Available URLs

/search/[term] search photos for particular term, displays 10 results
/search/[term]/[maximum] search photos for [term], displays [maximum] results
/api/search/[term] search photos for [term], displays 10 results serialized as JSON

Examples

/search/San%20Francisco/3 search for "San Francisco" and return 3 results
/api/search/San%20Francisco search for "San Francisco" and return json
